API Documentation

Available Endpoints

Listing Amenities

Endpoint: /api/listing-amenities/

Description: Retrieve a list of amenities associated with listings.

Listing Images

Endpoint: /api/listing-images/

Description: Access images related to property listings.

Listings

Endpoint: /api/listings/

Description: View and manage property listings.

Amenities

Endpoint: /api/amenities/

Description: Retrieve a list of all available amenities.

Bookings

Endpoint: /api/bookings/

Description: Manage property bookings and reservations.

Reviews

Endpoint: /api/reviews/

Description: Access and manage user reviews for properties.

Messages

Endpoint: /api/messages/

Description: Handle communication between users.

Wishlists

Endpoint: /api/wishlists/

Description: Manage user wishlists for favorite properties.

Locations

Endpoint: /api/locations/

Description: Retrieve information about various locations.
